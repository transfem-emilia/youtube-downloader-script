# How to Install yt-dlp

## Windows 11

1. Open Terminal
2. Type ``` winget install yt-dlp.yt-dlp ```
3. Follow all prompts.
4. Type ``` winget install yt-dlp.FFmpeg ```
5. Follow all prompts.

You have now installed yt-dlp and required dependencies.

## Windows 10

1. Open Windows Powershell
2. Type ``` winget install yt-dlp.yt-dlp ```
3. Follow all prompts.
4. Type ``` winget install yt-dlp.FFmpeg ```
5. Follow all prompts.

You have now installed yt-dlp and required dependencies.
